# Kryvexis Showcase Stable

A **compile-clean, Vercel-friendly** SaaS showcase for demos.

## What it is
- Apple-clean product shell
- Inbox
- Contacts CRM
- Automations
- Analytics
- Quotes
- Products
- Settings / team overview

## What it is not
- No external APIs yet
- No Supabase needed
- No env vars needed
- Data persists in the browser via localStorage

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
Just push to GitHub and import in Vercel.
No environment variables are required.

## Demo flow
1. Open Inbox
2. Click **Inject demo lead**
3. Reply to messages
4. Add automation rules
5. Show quotes and products
6. Explain that WhatsApp/API integration is the next phase
